Sphere Dimensions

Original Slip Question(s):

  • Write a Java program to accept dimensions of a sphere and print the surface area and volume.
import java.util.Scanner;

public class Sphere {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter radius of the sphere: ");
        double radius = sc.nextDouble();

        double surfaceArea = 4 * Math.PI * radius * radius;
        // Use 4.0/3.0 to prevent integer division
        double volume = (4.0 / 3.0) * Math.PI * Math.pow(radius, 3);

        System.out.println("Surface Area: " + surfaceArea);
        System.out.println("Volume: " + volume);
    }
}
Expected Output
Enter radius of the sphere: 5 Surface Area: 314.1592653589793 Volume: 523.5987755982989

Sort Employee Names

Original Slip Question(s):

  • Write a java program to accept n employee names from user. Sort them in ascending order and Display them.
import java.util.Scanner;
import java.util.Arrays;

public class SortNames {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter number of employees: ");
        int n = sc.nextInt();
        sc.nextLine(); // Consume newline

        String[] names = new String[n];
        for (int i = 0; i < n; i++) {
            System.out.print("Enter name " + (i + 1) + ": ");
            names[i] = sc.nextLine();
        }

        Arrays.sort(names); // Built-in sorting

        System.out.println("\nSorted Names:");
        for (String name : names) {
            System.out.println(name);
        }
    }
}
Expected Output
Enter number of employees: 3 Enter name 1: Charlie Enter name 2: Alice Enter name 3: Bob Sorted Names: Alice Bob Charlie

Interfaces & Abstract Shapes

Addresses these slip questions:

  • Define an interface "Operation" which has methods area(), volume()...
  • Define an abstract class Shape with abstract methods area() and volume()...
  • Define an Interface Shape with abstract method area()... (use final keyword).
interface Operation {
    // Variables in interface are inherently public static final
    final double PI = 3.142; 
    
    void area();
    void volume();
}

class Circle implements Operation {
    double radius;
    Circle(double r) { this.radius = r; }

    public void area() {
        System.out.println("Area of Circle: " + (PI * radius * radius));
    }
    public void volume() {
        System.out.println("Circle has no volume.");
    }
}

class Cylinder implements Operation {
    double radius, height;
    Cylinder(double r, double h) {
        this.radius = r;
        this.height = h;
    }

    public void area() {
        System.out.println("Surface Area of Cylinder: " + (2 * PI * radius * (radius + height)));
    }
    public void volume() {
        System.out.println("Volume of Cylinder: " + (PI * radius * radius * height));
    }
}

public class ShapeMain {
    public static void main(String[] args) {
        Circle c = new Circle(5);
        c.area();
        c.volume();
        
        System.out.println();
        
        Cylinder cy = new Cylinder(4, 10);
        cy.area();
        cy.volume();
    }
}
Expected Output
Area of Circle: 78.55 Circle has no volume. Surface Area of Cylinder: 351.904 Volume of Cylinder: 502.72

Employee/Manager (Array of Objects & Inheritance)

Addresses these slip questions:

  • Define class Employee (id, name, salary)... Subclass Manager with private member bonus... display worker having max total salary.
  • Write a program which define class Employee... Store information of 'n' employees...
import java.util.Scanner;

class Employee {
    int id;
    String name;
    double salary;

    void accept(int id, String name, double salary) {
        this.id = id;
        this.name = name;
        this.salary = salary;
    }
    void display() {
        System.out.print("ID: " + id + ", Name: " + name + ", Base Salary: " + salary);
    }
}

class Manager extends Employee {
    private double bonus;

    void acceptManager(int id, String name, double salary, double bonus) {
        super.accept(id, name, salary);
        this.bonus = bonus;
    }
    double getTotalSalary() {
        return salary + bonus;
    }
    void display() {
        super.display();
        System.out.println("\nTotal Salary (with bonus): " + getTotalSalary());
    }
}

public class EmployeeMain {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter number of managers: ");
        int n = sc.nextInt();
        Manager[] managers = new Manager[n];
        int maxIndex = 0;

        for (int i = 0; i < n; i++) {
            managers[i] = new Manager();
            System.out.println("Enter ID, Name, Salary, Bonus for Manager " + (i+1) + ":");
            managers[i].acceptManager(sc.nextInt(), sc.next(), sc.nextDouble(), sc.nextDouble());
            
            if (managers[i].getTotalSalary() > managers[maxIndex].getTotalSalary()) {
                maxIndex = i;
            }
        }
        System.out.println("\n--- Highest Paid Manager ---");
        managers[maxIndex].display();
    }
}
Expected Output
Enter number of managers: 2 Enter ID, Name, Salary, Bonus for Manager 1: 101 Alice 50000 5000 Enter ID, Name, Salary, Bonus for Manager 2: 102 Bob 60000 10000 --- Highest Paid Manager --- ID: 102, Name: Bob, Base Salary: 60000.0 Total Salary (with bonus): 70000.0

Custom Exceptions for Numbers

Addresses these slip questions:

  • Accept number... if zero throw "Number Is Zero", otherwise calculate sum...
  • Accept number... zero throw "Number is Zero". If non-numeric generate error "Number is Invalid" otherwise check palindrome.
  • Accept number... greater than 100 throw "Number is out of Range"...
import java.util.Scanner;

class ZeroException extends Exception {
    public ZeroException(String msg) { super(msg); }
}
class OutOfRangeException extends Exception {
    public OutOfRangeException(String msg) { super(msg); }
}

public class NumberValidation {
    static void processNumber(int num) throws ZeroException, OutOfRangeException {
        if (num == 0) {
            throw new ZeroException("Number Is Zero");
        } else if (num > 100) {
            throw new OutOfRangeException("Number is out of Range");
        } else {
            // Check Palindrome Logic
            int temp = num, rev = 0;
            while (temp > 0) {
                rev = (rev * 10) + (temp % 10);
                temp /= 10;
            }
            if (num == rev) System.out.println(num + " is a Palindrome.");
            else System.out.println(num + " is NOT a Palindrome.");
        }
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter a number: ");
        
        try {
            // Handles the "Non-numeric" invalid data requirement natively
            int num = Integer.parseInt(sc.nextLine());
            processNumber(num);
        } catch (NumberFormatException e) {
            System.out.println("Error: Number is Invalid (Non-numeric)");
        } catch (ZeroException | OutOfRangeException e) {
            System.out.println("User Defined Exception Caught: " + e.getMessage());
        }
    }
}
Expected Output
// Scenario 1: Valid Palindrome Enter a number: 121 121 is a Palindrome. // Scenario 2: Testing Custom ZeroException Enter a number: 0 User Defined Exception Caught: Number Is Zero // Scenario 3: Testing Non-numeric validation Enter a number: abc Error: Number is Invalid (Non-numeric)

File Analysis (Lines, Words, Chars)

Original Slip Question(s):

  • Write a java program to count number of lines, words and characters from a given file.
import java.io.*;

public class FileCounter {
    public static void main(String[] args) {
        // Ensure 'sample.txt' exists in the same directory before running
        File file = new File("sample.txt");
        int chars = 0, words = 0, lines = 0;

        try (BufferedReader br = new BufferedReader(new FileReader(file))) {
            String line;
            while ((line = br.readLine()) != null) {
                lines++;
                chars += line.length();
                String[] wordList = line.trim().split("\\s+"); 
                if(wordList.length > 0 && !wordList[0].isEmpty()) {
                    words += wordList.length;
                }
            }
            System.out.println("Lines: " + lines);
            System.out.println("Words: " + words);
            System.out.println("Characters: " + chars);
        } catch (IOException e) {
            System.out.println("File error: " + e.getMessage());
        }
    }
}
Expected Output
// Assuming sample.txt has some content: Lines: 5 Words: 24 Characters: 142

Basic Number Threads

Addresses these slip questions:

  • Write a Java program to create a thread that prints the first 5 Fibonacci numbers.
  • Write a Java program to create a thread that checks whether a number is Armstrong or not. (Repeated 3 times)
  • Write a Java program to take a number... and print all odd numbers from 1 to n.(Use thread)
class FiboThread extends Thread {
    public void run() {
        int a = 0, b = 1;
        System.out.print("Fibonacci: " + a + " " + b);
        for (int i = 2; i < 5; i++) {
            int c = a + b;
            System.out.print(" " + c);
            a = b; b = c;
        }
        System.out.println();
    }
}

class ArmstrongThread extends Thread {
    int num;
    ArmstrongThread(int n) { this.num = n; }
    public void run() {
        int temp = num, sum = 0;
        while (temp != 0) {
            sum += Math.pow(temp % 10, 3);
            temp /= 10;
        }
        System.out.println("Armstrong Check (" + num + "): " + (sum == num));
    }
}

class OddThread extends Thread {
    int max;
    OddThread(int max) { this.max = max; }
    public void run() {
        System.out.print("Odds up to " + max + ": ");
        for(int i=1; i<=max; i+=2) {
            System.out.print(i + " ");
        }
        System.out.println();
    }
}

public class BasicThreads {
    public static void main(String[] args) {
        new FiboThread().start();
        new ArmstrongThread(153).start(); // 153 is Armstrong
        new OddThread(9).start();
    }
}
Expected Output
// Note: Output order may vary because threads run simultaneously. Fibonacci: 0 1 1 2 3 Armstrong Check (153): true Odds up to 9: 1 3 5 7 9

Synchronization (Railway Reservation)

Original Slip Question(s):

  • Write a Java program to demonstrate multithreading using a Railway Reservation System, where one thread handles ticket booking and another thread handles ticket cancellation simultaneously.
class TicketBooking {
    int availableTickets = 10;

    // synchronized prevents race conditions
    synchronized void bookTicket(int requested) {
        if (availableTickets >= requested) {
            System.out.println(Thread.currentThread().getName() + " booked " + requested + " ticket(s).");
            availableTickets -= requested;
        } else {
            System.out.println("Tickets not available for " + Thread.currentThread().getName());
        }
        System.out.println("Remaining: " + availableTickets);
    }

    synchronized void cancelTicket(int amount) {
        availableTickets += amount;
        System.out.println(Thread.currentThread().getName() + " cancelled " + amount + " ticket(s).");
        System.out.println("Remaining: " + availableTickets);
    }
}

public class RailwaySystem {
    public static void main(String[] args) {
        TicketBooking system = new TicketBooking();

        Thread t1 = new Thread(() -> system.bookTicket(4), "User1(Book)");
        Thread t2 = new Thread(() -> system.cancelTicket(2), "User2(Cancel)");
        Thread t3 = new Thread(() -> system.bookTicket(5), "User3(Book)");

        t1.start();
        t2.start();
        t3.start();
    }
}
Expected Output
// Note: Output order depends on OS Thread Scheduler User1(Book) booked 4 ticket(s). Remaining: 6 User3(Book) booked 5 ticket(s). Remaining: 1 User2(Cancel) cancelled 2 ticket(s). Remaining: 3

Basic GUI & Action Listeners

Addresses these slip questions:

  • JFrame window with title "My First Swing Application" with JLabel that shows your name.
  • JFrame with a JButton that displays a message when clicked.
  • JLabel that changes color when a button is clicked.
import javax.swing.*;
import java.awt.*;
import java.awt.event.*;

public class BasicGUI {
    public static void main(String[] args) {
        JFrame frame = new JFrame("My First Swing Application");
        frame.setSize(300, 200);
        frame.setLayout(new FlowLayout());
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

        JLabel label = new JLabel("Hello, Student Name!");
        JButton btn = new JButton("Click Me");

        btn.addActionListener(new ActionListener() {
            public void actionPerformed(ActionEvent e) {
                JOptionPane.showMessageDialog(frame, "Button was clicked!");
                label.setForeground(Color.RED); // Change color requirement
            }
        });

        frame.add(label);
        frame.add(btn);
        frame.setVisible(true);
    }
}
Expected GUI Output
► A graphical window titled "My First Swing Application" opens. ► It contains a label "Hello, Student Name!" and a "Click Me" button. ► When the button is clicked: A popup alert says "Button was clicked!" and the original label text turns RED.

Combo Boxes & Radio Buttons

Addresses these slip questions:

  • GUI by using swing component JComboBox. On click of show button display selected language on Jlabel.
  • Create JRadioButtons for choosing payment mode... display selected mode when button clicked.
import javax.swing.*;
import java.awt.*;

public class ControlsGUI {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Input Controls");
        frame.setSize(350, 250);
        frame.setLayout(new FlowLayout());
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

        // --- Language Combo Box ---
        String[] langs = {"Java", "Python", "C++"};
        JComboBox<String> combo = new JComboBox<>(langs);
        JLabel langLbl = new JLabel("Lang: None");
        JButton langBtn = new JButton("Show Lang");
        langBtn.addActionListener(e -> langLbl.setText("Lang: " + combo.getSelectedItem()));

        // --- Payment Radio Buttons ---
        JRadioButton rbCash = new JRadioButton("Cash");
        JRadioButton rbUPI = new JRadioButton("UPI");
        ButtonGroup bg = new ButtonGroup(); 
        bg.add(rbCash); bg.add(rbUPI);
        
        JButton payBtn = new JButton("Pay");
        JLabel payLbl = new JLabel("Mode: None");

        payBtn.addActionListener(e -> {
            if (rbCash.isSelected()) payLbl.setText("Mode: Cash");
            else if (rbUPI.isSelected()) payLbl.setText("Mode: UPI");
        });

        frame.add(combo); frame.add(langBtn); frame.add(langLbl);
        frame.add(new JSeparator(SwingConstants.HORIZONTAL));
        frame.add(rbCash); frame.add(rbUPI); frame.add(payBtn); frame.add(payLbl);

        frame.setVisible(true);
    }
}
Expected GUI Output
► A graphical window titled "Input Controls" opens. ► Action 1: User selects "Python" from the dropdown and clicks "Show Lang". The label changes to "Lang: Python". ► Action 2: User clicks the "UPI" radio button and clicks "Pay". The label changes to "Mode: UPI".

Mouse Events (Listener)

Original Slip Question(s):

  • Design a screen in Java to handle the Mouse Events such as MOUSE_MOVED and MOUSE_CLICK and display the position of the Mouse_Click in a TextField.
import javax.swing.*;
import java.awt.*;
import java.awt.event.*;

public class MouseEventsGUI extends JFrame implements MouseListener, MouseMotionListener {
    JTextField tf;

    MouseEventsGUI() {
        setTitle("Mouse Events");
        setSize(400, 300);
        setLayout(new FlowLayout());
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

        tf = new JTextField(25);
        add(tf);

        // Add listeners to track mouse over the frame
        addMouseListener(this);
        addMouseMotionListener(this);
        
        setVisible(true);
    }

    public void mouseMoved(MouseEvent e) {
        tf.setText("Moved to X:" + e.getX() + " Y:" + e.getY());
    }
    public void mouseClicked(MouseEvent e) {
        tf.setText("Clicked at X:" + e.getX() + " Y:" + e.getY());
    }
    
    // Required empty methods from interfaces
    public void mouseDragged(MouseEvent e) {} 
    public void mousePressed(MouseEvent e) {}
    public void mouseReleased(MouseEvent e) {}
    public void mouseEntered(MouseEvent e) {}
    public void mouseExited(MouseEvent e) {}

    public static void main(String[] args) {
        new MouseEventsGUI();
    }
}
Expected GUI Output
► A window with a single text field appears. ► Moving mouse inside window: Text field rapidly updates to "Moved to X:150 Y:120", etc. ► Clicking mouse inside window: Text field updates to "Clicked at X:150 Y:120".
